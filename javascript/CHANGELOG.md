## [2.5.0] - 2025-11-28

### Added
- Initial npm release of JewelMusic JavaScript/TypeScript SDK
- AI Copilot resource for melody, harmony, and lyrics generation
- Music Analysis resource for audio quality checking and structure detection
- Distribution resource for managing releases across streaming platforms
- Transcription resource with 150+ language support
- Analytics resource for streaming data and royalty tracking
- User management resource
- Webhooks resource with signature verification
- Full TypeScript support with type definitions
- Browser and Node.js environment support
- Automatic retry with exponential backoff
- Comprehensive error handling with specific error types
