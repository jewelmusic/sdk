# Changelog

All notable changes to the JewelMusic Python SDK will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.2] - 2025-11-28

- Fixed broken relative links in README for PyPI display

## [1.0.1] - 2025-11-28

### Fixed
- Fixed package name from `jewelmusic-sdk` to `jewelmusic`
- Updated PyPI badges to use correct package name
- Fixed GitHub repository links

## [1.0.0] - 2025-11-28

### Added
- Initial release of JewelMusic Python SDK
- AI Copilot resource for melody, harmony, and lyrics generation
- Music Analysis resource for audio quality checking and structure detection
- Distribution resource for managing releases across streaming platforms
- Transcription resource with 150+ language support
- Analytics resource for streaming data and royalty tracking
- User management resource
- Webhooks resource with signature verification
- Full async/await support with optional synchronous interface
- Comprehensive error handling with specific exception types
- Type hints throughout the codebase
- Context manager support for automatic cleanup
